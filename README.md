# spring-restapi-hibernate-customer-order

## Used technologies:
- Spring Boot
- Spring Data JPA
- Lombok
- Mysql Database
- Hibernate

## Requirements
For building and running the application you need:
- JDK 1.8 or newer
- Lombok
- Maven

## Port
```
  http://localhost:8080
```
